## Description

The following project was made for the **Udacity's Front End Web Developer Nanodegree**.
It is based on the frogger arcade game, which objective is to reach the water.

## Instructions

### Installation

- To download the game you can visit the following website: [Frogger Game](https://github.com/JulioVL/frontend-nanodegree-arcade-game).
- Clone it and open index.html
```
$ git clone https://github.com/JulioVL/frontend-nanodegree-arcade-game.git
```
- Or you can try it on this site: [Frogger Game](https://juliovl.github.io/frontend-nanodegree-arcade-game/).

### Game Instructions

1. Use the arrow keys to move across the tiles.
2. Avoid the bugs, they will reset your player position and score.
3. The objective is to obtain 200 points.
4. Reaching the water gives you 10 points.
5. Every gem gives you 50 points.
6. To restart the game, press the "enter" key.
